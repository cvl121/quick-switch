# Quick Switch

A lightweight macOS menu bar app for changing your default browser in one click.

Instead of opening System Settings or going through the browser menu every time you want to change your default browser, Quick Switch puts a dropdown in your menu bar with all your installed browsers listed. Click one to make it your new default — no confirmation dialogs, no extra steps.

## Features

- **Menu bar icon** shows your current default browser
- **One-click switching** between any installed browser
- **Auto-detection** of installed browsers, with live monitoring of `/Applications` for new installs
- **Launch at Login** option to keep it running in the background
- **No dock icon** — runs quietly in the menu bar

## Requirements

- macOS 13.0 (Ventura) or later

## Tested On

- macOS 26.3.1 (Tahoe), Apple Silicon

## Install

1. Download `QuickSwitch-1.0.0.dmg` from the [Releases](https://github.com/cvl121/quick-switch/releases) page
2. Open the DMG
3. Drag **Quick Switch** into the **Applications** folder
4. Launch from Applications

## Usage

Once launched, a browser icon appears in your menu bar. Click it to see a list of your installed browsers. The currently active default has a checkmark next to it. Click any other browser to switch.

The menu also includes:

- **Refresh Browsers** — rescan for installed browsers
- **Launch at Login** — toggle starting Quick Switch on login
- **Quit** — stop the app
